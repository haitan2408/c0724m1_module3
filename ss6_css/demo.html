HTML (HyperText Markup Language) là ngôn ngữ đánh dấu tiêu chuẩn được sử dụng để tạo và cấu trúc nội dung trên các trang web. HTML không phải là ngôn ngữ lập trình, mà là ngôn ngữ đánh dấu để mô tả cấu trúc và trình bày thông tin trên Internet. Từ các trang web đơn giản đến những ứng dụng web phức tạp, HTML là nền tảng không thể thiếu trong phát triển web.
